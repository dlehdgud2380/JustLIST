# py_JustLIST 
그냥 예제보고 하면서 공부하기엔 정말 재미도 없고해서 '4.1 Structured types 1.pptx' 즉 리스트와 관련된 예제란 예제란 예제는 갈아넣어서 아주 기초적인 프로그램을 만들어보았다. 
이름하여 'Just LIST'

**※ ONLY WORKING Python 3 not guarantee work Python 2.x**

##기능:
1. item값 추가
2. item값 수정
3. LIST 자른 결과값 출력
4. item값 삭제
5. LIST에서 item찾기
6. LIST 정렬하기
7. LIST값 상태
8. Just List 정보
9. Just LIST 새로시작
